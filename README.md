# 🛒 Nova Mart

Nova Mart is a responsive e-commerce product catalog web application built using HTML, CSS, and JavaScript. The application fetches product data from an external API and provides a seamless shopping experience with filtering, sorting, cart management, and dark mode support.

# 🚀 Features

- Fetches products dynamically from DummyJSON API
- Product filtering by category, price, and rating
- Product sorting options
- Shopping cart with quantity management
- Local Storage support for cart persistence
- Dark Mode toggle
- Toast notifications
- Responsive design for mobile, tablet, and desktop
- No results handling
- Smooth UI animations and interactions

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Local Storage
